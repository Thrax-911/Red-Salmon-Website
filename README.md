## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Thrax-911/Red-Salmon-Website.git
````

2. Navigate to the project folder

```bash
cd Red-Salmon-Website
```

3. Open the project in your browser

```
Open index.html
```

Or run it using **Live Server** in VS Code.

## 🚀 Usage

1. Open the website in your browser.
2. Explore the layout and interface.
3. Modify HTML, CSS, or JavaScript files to customize the design.

## 🛠 Troubleshooting

**Problem:** Website styles not loading
**Solution:**
Make sure the CSS path is correct.

Example:

```html
<link rel="stylesheet" href="css/style.css">
```
